# Employee Attendance Load Testing System

A full-stack employee attendance management application built with React, Node.js, Express, and MySQL.

## Features
- Employee attendance marking
- MySQL database integration
- Admin dashboard
- Attendance history
- Server monitoring page
- Load testing simulation for 50 employees

## Tech Stack
- React.js
- Node.js
- Express.js
- MySQL
- Git & GitHub
- AWS (Future Deployment)

## Project Goal

This project simulates an organization with 50 employees. During peak hours, 30+ employees submit attendance simultaneously to analyze:

- Server response time
- Database performance
- CPU and memory utilization
- Bottlenecks in backend processing
